### 기능
##### 노트
- [ ] 노트 목록 조회 형식 수정(카드 형식, 한국어 100자까지)
- [ ] 노트-표현 ManyToMany 풀기, 순서 저장
- [ ] 표현 순서 변경
- [ ] 노트 상세조회
- [ ] 노트 삭제
##### 표현
- [ ] 표현 목록 조회
- [ ] 표현 생성
- [ ] 표현 생성
- [ ] 표현 삭제
##### 로그인
- [ ] 로그인
##### 추가기능
- [ ] [노트 작성] 사용할 수 있는 표현 추천
- [ ] [노트 작성] 기존에 사용한 표현 불러오기

### 메모
- 노트 - 카테고리, 댓글, 좋아요

- 통계 - 일간, 월간, 년간 노트수, 표현수
